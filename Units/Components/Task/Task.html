Давайте ознакомимся с тренажёром
  1. Прочитайте условие задачи
  2. Решите задачу и впишите в поле для проверки ответа, полученный вами ответ без единиц измерения и других слов, только число.
  3. Проверка ответов осуществляется в кноце теста
  4. Происходит вывод количества правельно введённых ответов
  5. Если ответ прошёл проверку и оказался верным, то поле ввода загорается зелёным цветом, а также появляется кнопка "Показать решение", нажав на которую у вас есть возможность увидеть правильное решение
  6. Если обновите страницу у Вас будет тоже задание, но с другими числовыми данными.
  7. При каждом обновлении страницы числовые данные меняются
  8. Ведётся учёт времени
  9. Возвращаться к пропущенным заданиям нельзя
